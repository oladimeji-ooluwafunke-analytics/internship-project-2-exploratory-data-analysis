# Internship Project 2: Exploratory Data Analysis (EDA)

## Project Overview

This project focused on performing Exploratory Data Analysis (EDA) using Microsoft Excel. 
The objective was to examine the cleaned dataset, calculate descriptive statistics, identify trends, detect outliers,
and summarize meaningful observations from the data.

## Objectives

- Analyze the cleaned dataset.
- Calculate descriptive statistics.
- Identify trends and patterns.
- Detect outliers.
- Summarize key observations.

## Tools Used

- Microsoft Excel

## Dataset

The dataset contains 1,200 sales transaction records with the following fields:

- OrderID
- Date
- CustomerID
- Product
- Quantity
- UnitPrice
- ShippingAddress
- PaymentMethod
- OrderStatus
- TrackingNumber
- ItemsInCart
- CouponCode
- ReferralSource
- TotalPrice

## Analysis Performed

### Descriptive Statistics

Calculated:

- Count
- Mean
- Median
- Minimum
- Maximum

For the following columns:

- Quantity
- UnitPrice
- ItemsInCart
- TotalPrice

### Pivot Table Analysis

Created PivotTables to answer business questions:

- Total revenue by Product
- Number of orders by Payment Method
- Number of orders by Order Status
- Total revenue by Referral Source

### Outlier Analysis

Created a Box & Whisker Chart to identify outliers in the TotalPrice distribution.

### Summary

Documented the key findings and business insights from the analysis.
